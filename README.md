# Анализатор отзывов — Telegram-бот

Telegram-бот, который анализирует отзывы о товарах с Wildberries с помощью нейросети GigaChat.

## Проблема

При выборе товара в интернет-магазине сложно быстро понять, хороший ли он. У товара может быть тысячи отзывов, и прочитать все невозможно. Бот решает эту проблему — он скачивает отзывы и кратко рассказывает о плюсах, минусах и даёт рекомендацию.

## Как запустить

1. Клонируйте репозиторий:
```
git clone https://github.com/vikks0/AlShoppingAssistant.git
cd AlShoppingAssistant
```

2. Создайте и активируйте виртуальное окружение:
```
python3 -m venv .venv
source .venv/bin/activate
```
*На Windows: `.venv\Scripts\activate`*

3. Установите зависимости:
```
pip install -r requirements.txt
```

4. Создайте файл `.env` и добавьте токены:
```
BOT_TOKEN=ваш_токен_бота
GIGACHAT_API_KEY=ваш_ключ_API
```

5. Запустите бота:
```
python bot.py
```

## Как работает

1. Пользователь отправляет ссылку на товар с Wildberries
2. Бот загружает Selenium и собирает все отзывы
3. Отзывы отправляются в GigaChat для анализа
4. Бот возвращает: количество отзывов, плюсы, минусы, рекомендацию

## Вход и выход

**Вход:** ссылка на товар с Wildberries

**Выход:**
- Количество положительных и отрицательных отзывов
- Плюсы товара
- Минусы товара
- Рекомендация к покупке

## Технологии

- **Python 3.8** — основной язык
- **pyTelegramBotAPI** — работа с Telegram
- **Selenium** — загрузка отзывов с Wildberries
- **GigaChat (Sber)** — анализ отзывов с помощью ИИ
- **SQLite** — хранение истории запросов
- **python-dotenv** — работа с переменными окружения

## Использование ИИ

Бот использует GigaChat (Sber) для анализа отзывов. Нейросеть получает список отзывов и выдаёт структурированный ответ: плюсы, минусы и рекомендацию. Промпт настроен так, чтобы ответ был кратким и без лишней информации.

## Ограничения

- Работает только с Wildberries (Ozon и AliExpress из-за антибот-защиты)
- Wildberries блокирует автоматизированный доступ — не все товары загружаются
- Wildberries периодически обновляет HTML-вёрстку и CSS-классы. Парсер использует конкретные селекторы (`.comments__item.feedback`, `[class*='feedback__text--item']`), которые могут устареть — тогда бот перестанет находить отзывы и потребует правки `parser.py`
- Анализ зависит от качества модели GigaChat
